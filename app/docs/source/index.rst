Install
-----------------------------

**Requirements**

Offene Wahlen AT API officially supports Python 3.7.

Python packages:

- FastAPI==0.61.2
- uvicorn==0.12.2
- SQLAlchemy==1.3.20
- tqdm==4.50.2
- python-dotenv==0.14.0
- psycopg2
- typer==0.3.2
